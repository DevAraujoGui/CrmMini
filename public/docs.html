LeadBoard REST Console

O console interativo da API do LeadBoard permite disparar requisições diretamente para as rotas Express rodando localmente no banco de dados SQLite.

$ curl -X GET http://localhost:3000/api/health
# status: 200 OK | database: connected
$ node --version # v20.x SQLite active
