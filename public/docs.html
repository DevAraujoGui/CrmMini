Bem-vindo ao Playground Interativo da API do CRMini! Esta página funciona de maneira semelhante ao Swagger UI, listando todos os endpoints disponíveis na sua aplicação e permitindo que você os teste ao vivo diretamente no banco de dados SQLite local.

Para rotas que exigem um ID (ex: DELETE /api/leads/:id), insira o ID correspondente na caixa correspondente. Para rotas que requerem dados (como cadastro e login), você pode modificar o corpo (Body JSON) livremente antes de testar.
